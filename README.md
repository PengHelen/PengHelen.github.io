## run the project

+ install hexo 

`npm install -g hexo-cli`

`npm install hexo`

+ init & run 

`hexo server`

## build & deploy

`hexo clean`

`hexo generate`

`hexo deploy`

# init menu

`hexo new page "menu's name"`

# write article

+ new article

`hexo new <title>`
